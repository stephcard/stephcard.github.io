Project background

CNN's civic engagement platform, CITIZEN by CNN, was scheduled to host an activation, Citizen House @ SXSW, from March 14-15, 2020.

Considering the high volume of foot traffic at SXSW, the need to keep audiences engaged while waiting to enter the activation emerged.

My role was to produce political trivia games. The games were to be installed in iPad kiosks to keep audiences engaged while they waited.

Due to COVID-19, SXSW was canceled, but screen recordings of CITIZEN by CNN Game Pack live on in my portfolio.

Project requirements

The CITIZEN by CNN Game Pack should include three to four highly engaging games.

The games should be short and thoughtfully designed, so audiences are inclined to play multiple times.

The games should feature meaningful content, so audiences feel informed and educated.

Process recap

After brainstorming game concepts, a user survey was conducted to identify which ideas resonated the most with audiences ages 21-54, SXSW's demographic.

Next, paper wireflows would identify content for each game and begin to outline the user flows. User testing was conducted to make sure I was on the right track.

Content for each game concept was collected and organized into a spreadsheet that could be exported as a JSON to populate the game interface.

With a better understanding of the content, the second round of paper wireflows solidified user flows. User testing revealed game constraints, such as time limits, increased audience interest.

Members only a new kind of house party. Players identify if the congressperson is a member of the House or Senate. Within a time limit of sixty seconds, players answer as many congresspersons correctly as possible for the best score. The presented congresspersons are randomized - players will not get the same congresspersons if playing a second time or with a friend.

Who said what? Politics edition. Players identify which political figure said the presented quote. The game interface presents quotes with audio via a headset provided at the iPad kiosk. The presented quotes are randomized - players will not get the same quotes if playing a second time or with a friend. There are ten presented quotes per play.

Which candidates ran in 2020? Players answer who were the presidential candidates in 2020. The game presents thirty-five candidates, but only thirty-two are correct. Tap on an incorrect candidate, and it's game over.

We want you! Try to pass the US citizenship exam. Players answer US citizenship exam questions. The game presents randomized questions - players will not get the same questions if playing a second time or with a friend. There are ten questions presented per play.

High-fidelity prototypes were produced based on the wireflows and used in guerilla usability testing to validate the user flows and expectations.

Project background

To increase user registration across CNN.com, I was tasked to produce a politics trivia game that would live as a standalone app.

Project requirements

The game should be free to play but registration is required for certain parts of the game.

Annotated wireflows were to be delivered in two weeks.

Process recap

A competitive research and analysis of similar experiences and products was conducted. This exercise included a lot of screengrabs with observation notes and a competitive matrix.

Metrics on game-like experiences produced by CNN were pulled to understand how they performed with our audiences.

User tasks and requirements are outlined.

Paper wireframes are sketched to begin to flesh out the required screen for the experience.

Guerilla usability testing was conducted on the paper wireframes to ensure the screens meet user expectations.

Paper wireframes evolved into wireflows, which outline user flows with more context than a wireframe.

Another round of guerilla usability testing validates if user tasks can be completed successfully.
